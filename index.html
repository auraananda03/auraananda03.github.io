🌵About Me🌵

Hallo semua! Nama saya Aura Ananda Haryanto. Biasa dipanggil Nanda, tapi ada beberapa orang yang manggil saya Aura, gapapa sih bebas aja mau manggil apa hehe. Saya lahir di Yogyakarta, 3 September 2001. Banyak yang gak percaya kalo saya orang Yogja katanya sih gak kemayu kayak orang-orang Yogja disana. Saya lahir di Yogya tapi saya dibesarkan di Cibinong. Saya anak pertama dari 2 bersaudara. Ayah saya bekerja sebagai wiraswasta dan ibu saya bekerja sebagai ibu rumah tangga. Saya tinggal di Padurenan RT 01 RW 01 Kelurahan Pabuaran Kecamatan Cibinong Kabupaten Bogor. Saya sudah lama tinggal di tempat ini kurang lebih selama 13 tahun. Sekarang umur saya masih 16 tahun tahun 2018 ini umur saya akan bertambah menjadi 17 tahun.

Saya bersekolah di SMK-SMAK Bogor, kelas 11 atau 2 SMK. Saya sekolah jurusan Analis Kimia. Cita-cita saya menjadi Dokter Anak. Saya ingin sekali melanjutkan pendidikan di Universitas Indonesia. Saya berencana ingin masuk jurusan kedokteran atau teknik kimia. Doain ya teman-teman!

Hobi saya yaitu shopping, menyanyi, menonton film dan bermain basket.Saya sangat suka shopping di mall untuk membeli barang barang baru dan menarik buat saya. Saya sangat suka menyanyi apalagi kalu lagi sendiri dirumah hehe. Saya juga ikut eskul paduan suara di sekolah bersama teman teman yang lainnya. Saya senang sekali menonton film khususnya drama korea, sudah banyak drama korea yang saya tonton dan seru-seru semua. Saya senang sekali bermain basket, dan saya bergabung dalam tim basket di angkatan saya.
